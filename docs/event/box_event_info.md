* 售卖地址每次<font color="red">**不固定**</font>，会提前五分钟左右在群里通知！！～每次售卖地址不同！<font color="red">**请勿**</font>提前排队内卷！按照箭头位置自觉排队！
    
## 购买方式

* 盲盒购买方式：交易:star:售卖人员:star:,告知想要购买的盲盒号码，**获得信物**即可！

* 明盒购买方式：告知售卖人员心仪的:star:太太的ID:star:， **获得信物**即可！

* 购买盲盒交易信物后，直接进入画室寻找太太给与**信物和所需金币**！太太会把客人组进队伍！**等待太太截下id 以及角色信息图**。等太太告知后可以自由活动，或者站在太太旁边。

* 由于是低价速写盲盒所以不接受任何画面要求指定，无论明盲都是 **一 键 出 图**。详情一切根据太太本人心理标准进行协商变动。

* 出图时间定位一周之内，但是**实际出图**时间一切以太太为主。

* 如果没有抽到自己心仪的太太，可以找太太私下沟通，询问太太是否有时间，协商合适的价格进行约稿。
