## 2.活动范围

* 目前画室活动基本在周六周日，具体时间会在**群公告**中提醒

## 3.活动内容

* 每次预约参加活动的太太会提前联系活动管理者给出想要参与的作画活动的张数相同数量的信物:gift:，玩家可以在指定时间指定地点排队从工具人:santa:手中抢购信物:gift:，并在之后拿着信物:gift:由太太完成作画交给购买的玩家。总而言之：画画！
    
## 4.画师资格的审核和申请

* 如有画师想申请画师资格一起快乐画画，可以进群联系**画室管理员**:heavy_check_mark:

* 每次活动的参与**非硬性要求**:heavy_check_mark:

* 所有画画相关的金额将全额从玩家手中交给画师太太，**没有中间商赚差价**:heavy_check_mark:

* 太太需要在画室规定的时间期限内完成相关作画，不然玩家支付的金额会**全额**退还给玩家:no_entry_sign:

* 本画室组织属于非盈利性质，画师的辛苦劳作需要玩家鼓励，参与活动购买信物需支付**小部分游戏币**，由于活动属游戏内活动，**拒不接受RMT**:x:，感谢玩家理解
